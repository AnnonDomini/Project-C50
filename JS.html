JavaScript is a scripting or programming language that allows you to implement complex features on web pages

  • JavaScript allows you to display timely content updates, interactive maps, animated 2D/3D graphics, scrolling video jukeboxes, etc.
  • JavaScript is the third layer of the layer cake of standard web technologies, two of which (HTML and CSS) we have covered in much more detail in other parts of the Learning Area
  • JavaScript allows developers to build complex interactive websites and browser games, and to connect servers to websites and web applications.
